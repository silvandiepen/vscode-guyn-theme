# Guyn
Guyn is a simple theme based on the Guyn Colors

## Installation
 1. Open Extensions on the sidebar
 2. Search for "**Guyn**"
 3. Install it
 4. Reload the editor
 5. Press ⌘K + ⌘T
 6. Pick "**Guyn**" one from the list and hit enter
 7. Enjoy!
